## 📂 Vault Structure

We use a flat, highly scalable **4-Folder System** to keep your sidebar clean:

1. **`00 Inbox`**: The landing pad. All new quick notes, fleeting thoughts, and web captures go here. Process this folder daily to keep it empty.
2. **`10 Notes`**: Your permanent knowledge base. This is where your structured daily logs, project outlines, and research notes live.
3. **`20 Templates`**: Blueprint files. Pre-formatted notes that you can instantly insert into new pages to save time (Daily Notes, Projects, Meetings).
4. **`90 System`**: Under the hood. Holds your media assets (`91 Attachments`) and custom CSS snippets.

## ⚡ Essential Hotkeys

Get comfortable using your keyboard to navigate Obsidian at lightspeed:

| Command | Windows Shortcut | Action |
| --- | --- | --- |
| **Quick Switcher** | `Ctrl + O` | Find or create notes instantly |
| **Command Palette** | `Ctrl + P` | Search and run any Obsidian command |
| **New Note** | `Ctrl + N` | Create a blank page in the Inbox |
| **Insert Template** | *Via Command Palette* | Apply a note template layout |
| **Toggle Sidebar** | `Ctrl + H` | Collapse/expand the navigation folders |
